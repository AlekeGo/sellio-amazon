# Sellio — Frontend

React 19 + TypeScript + Vite frontend for Sellio.

See the project root [README.md](../README.md) for full setup and deployment instructions.

## Quick start

```bash
npm install
cp .env.example .env
npm run dev
```

## Build

```bash
npm run build
```

Output is written to `dist/`. Deploy `dist/` to Vercel or any static host.
SPA routing is configured in `vercel.json`.
